**Hyperspectral Image Classification using Spectral-Spatial Mixer Network**, Accepted for publication in IEEE WHISPERS Conference.

# Datasets:
Two Commonly used datasets were used in this paper, namely  QUH-Tangdaowan and QUH-Qingyun
<img width="788" height="767" alt="image" src="https://github.com/user-attachments/assets/0bb4ada5-e6f6-4dcd-a2df-7502cd74ad7e" />

# Requirement
Python 3.9.18, Tensorflow (and Keras) 2.10.0

# Results
To quantitatively measure the proposed SS-MixNet model, three evaluation metrics are employed to verify the effectiveness of the algorithm, Overall Accuracy (OA), Average Accuracy (AA) and Cohen's Kappa (k). Also, Each class accuracy has been reported
